💡 แนวคิดเพื่อการเรียนการสอน (Key ML Concepts for Students)

เมื่อผู้ใช้คลิกปุ่มทำนาย ข้อมูลฝั่งหน้าเว็บนี้จะถูกส่งไปที่เซิร์ฟเวอร์หลังบ้าน (Flask) เพื่อทำการประมวลผลตามขั้นตอน Pipeline ดังนี้:

  • Imputation & Scaling: ข้อมูลดิบจะถูกประมวลผ่าน [Preprocessor Pipeline] ที่บันทึกไว้ ตัวอย่างเช่น โมเดลจะแปลงค่าอายุและค่าตั๋วให้เป็น Standard Normal Vector โดยคำนวณจากค่าเฉลี่ยของข้อมูลจริงที่ศึกษาไว้ในตอนแรก
  • Decision Boundary: อัลกอริทึมจะนำจุดข้อมูลที่ถูกสเกลแล้วส่งต่อไปให้โมเดลเพื่อคำนวณความน่าจะเป็นสะสม (Probability) แล้วแปลออกมาเป็นรหัส 0 (เสียชีวิต) หรือ 1 (รอดชีวิต)
  • Real-time API Call: เป็นตัวอย่างให้นักศึกษาเข้าใจว่า โมเดล Machine Learning ในคอมพิวเตอร์สามารถนำมาเชื่อมโยงให้บริการภายนอกผ่าน Web API เพื่อสร้างนวัตกรรมแอปพลิเคชันจริงได้อย่างไร
